# 🌊 Acerca de Musify

**Musify** es una aplicación móvil enfocada en streaming musical online y reproducción offline, diseñada para brindar una experiencia moderna inspirada en plataformas musicales premium.

La aplicación fue desarrollada utilizando Flutter, permitiendo un rendimiento fluido, diseño responsive y compatibilidad multiplataforma.

Musify ofrece búsqueda online, reproducción de alta calidad, descargas locales y soporte avanzado para letras sincronizadas y formatos de audio de alta fidelidad.
